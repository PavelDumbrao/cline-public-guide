# TaskComplete Template

## Зачем нужен
TaskComplete hook полезен, если ты хочешь отправлять уведомления о завершении задачи.

## Как сделать безопасно
1. Не храни токен в самом скрипте.
2. Читай токен из переменной окружения.
3. Используй placeholders в документации.
4. Проверь, что stdout возвращает только JSON.

## Пример безопасной идеи
- `TELEGRAM_BOT_TOKEN` брать из env
- `TELEGRAM_CHAT_ID` брать из env
- если переменной нет — hook должен мягко завершаться без ошибки

## Что нельзя делать
- публиковать реальный токен
- публиковать реальный chat id
- смешивать логирование и JSON output
