# Hooks

Здесь лежат безопасные примеры hooks для Cline.

## Что включено
- `TaskStart`
- `PreToolUse`
- `PostToolUse`
- `TaskComplete_TEMPLATE.md`

## Что не включено как готовый script
`TaskComplete` с реальной отправкой во внешний сервис.

Причина: в публичном репозитории нельзя публиковать hooks с живыми токенами.
