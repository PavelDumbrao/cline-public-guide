# Browser Automation

> Этот файл — про **практическое использование** browser automation.
> Если тебе нужны ограничения и чувствительные сценарии, смотри также [`16-browser-automation-safety.md`](16-browser-automation-safety.md).

## Когда использовать
- Проверка интерфейса
- Поиск ошибок в консоли
- Проверка network-запросов
- Smoke test после изменений

## Правила
- Сначала изучи страницу
- Действуй по шагам
- Проверяй консоль и сеть
- Не делай выводы только по одному скриншоту
