# Browser Automation

> Этот файл — про **практическое использование** browser automation.
> Если тебе нужны ограничения и чувствительные сценарии, смотри также [`16-browser-automation-safety.md`](16-browser-automation-safety.md).

Сначала полезно прочитать:
- [`../../docs/18-cline-ui-map.md`](../../docs/18-cline-ui-map.md) — чтобы понимать, где browser-related вещи вообще живут в интерфейсе
- [`16-browser-automation-safety.md`](16-browser-automation-safety.md) — для границ безопасности

## Когда использовать
- Проверка интерфейса
- Поиск ошибок в консоли
- Проверка network-запросов
- Smoke test после изменений

## Правила
- Сначала изучи страницу
- Действуй по шагам
- Проверяй консоль и сеть
- Не делай выводы только по одному скриншоту

---

## ✅ Чек-лист для самопроверки
- [ ] Я использую browser automation только когда это реально нужно
- [ ] Я сначала смотрю страницу, а не сразу кликаю куда попало
- [ ] Я проверяю консоль и сеть, а не только картинку
- [ ] Я не делаю выводы по одному скриншоту без дополнительных сигналов
