# Tool Usage

## Принцип
Каждый инструмент должен использоваться осознанно.

## Правила
- Сначала читай, потом меняй.
- Перед рискованным действием понимай последствия.
- После изменения проверяй результат.
- Не объявляй задачу завершённой без фактической проверки.

## Коммуникация
Если чего-то не хватает — задай короткий точечный вопрос.
