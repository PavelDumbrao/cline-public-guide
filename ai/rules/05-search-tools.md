# Search Tools

## Принцип выбора инструмента
- Документация библиотеки → Context7
- Веб-поиск и research → Tavily / Exa
- Поиск по локальной кодовой базе → semantic/local code search
- Проверка UI → Playwright

## Правило
Не используй сложный инструмент, если хватает более простого.

## Проверка
После поиска перепроверь ключевой вывод минимум по одному надёжному источнику.
