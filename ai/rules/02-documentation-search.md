# Поиск документации

## Когда искать
- Если не знаешь точный API.
- Если нужна актуальная синтаксическая деталь.
- Если библиотека быстро меняется.
- Если пользователь просит сделать что-то в незнакомом инструменте.

## Приоритеты
1. Официальная документация
2. Официальные руководства
3. Авторитетные технические статьи
4. Проверенные примеры кода

## Правило
Не выдумывай конфигурацию библиотек, если её можно проверить по docs.
