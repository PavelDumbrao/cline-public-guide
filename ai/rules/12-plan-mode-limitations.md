# Plan Mode Limitations

## Принцип
В режиме планирования сначала собирай контекст, потом предлагай план.

## Правила
- Не обещай изменения, пока не проверил реальное состояние файлов.
- Если для выполнения нужны действия записи — попроси перейти в режим выполнения.
- План должен быть коротким, конкретным и выполнимым.
