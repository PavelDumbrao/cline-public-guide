# /install-skill

## Цель
Безопасно добавить новый skill в проект.

## Шаги
1. Определи, действительно ли skill нужен.
2. Проверь, нет ли уже похожего skill.
3. Создай папку `skill-name/`.
4. Создай `SKILL.md`.
5. Добавь: когда использовать, когда не использовать, ошибки, проверку результата.
6. Проверь, что внутри нет секретов.
