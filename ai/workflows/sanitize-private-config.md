# /sanitize-private-config

## Цель
Очистить рабочие артефакты перед публикацией.

## Чек-лист
- токены
- ключи
- chat ids
- email
- домены
- IP
- webhook URLs
- database URLs
- credentials blocks
- user-specific names

## Правило
Если есть сомнение — лучше заменить на placeholder или не публиковать.
