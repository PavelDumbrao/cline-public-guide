# /prepare-public-repo

## Цель
Собрать публичный репозиторий из живой рабочей системы.

## Шаги
1. Отделить "для человека" и "для AI".
2. Убрать секреты и личную инфраструктуру.
3. Не путать Cline Workflows и n8n workflows.
4. Оставить только reusable шаблоны.
5. Добавить README и docs для новичка.
6. Сделать финальный self-review.
