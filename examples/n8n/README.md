# n8n Examples

Здесь лежат учебные примеры workflow для n8n.

## Важно
Это **не Cline Workflows**.
Это JSON-файлы для импорта в n8n.

## Перед публикацией любой n8n схемы
Проверь:
- нет ли `credentials`
- нет ли токенов
- нет ли file IDs
- нет ли приватных URL
- понятно ли, что ученик должен заменить вручную
